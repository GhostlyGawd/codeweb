One graph, two interfaces

Built for the humans and the agents editing the code

codeweb computes one structural graph and serves it two ways — so a person and the AI agent working beside them finally share the same map.

For humans — an interactive map

A self-contained HTML report: the dependency web, duplication, hotspots, and ranked, cycle-safe consolidation opportunities. No CDN, no network, opens straight from disk.

For agents — 20 deterministic tools

Over MCP, a coding agent can ask what already does this, what an edit breaks, and which fix is safe — before it writes a line. No LLM in the analysis loop, so the answers don't drift.

How it works

A four-stage deterministic pipeline

Parse-free extraction on the fast path for JavaScript, TypeScript, Python, Rust, and Go, with an agent fallback for everything else.

01

extract

Parse-free atomic nodes + edges, per language (JS/TS/Python/Rust/Go).

02

cluster

Group nodes into semantic domains.

03

overlap

Body-confirm cross-domain duplication and rank it.

04

render

A self-contained interactive map + 20 agent tools.

Two ways to use it

Restructure your own code — or vet someone else's

Map your own codebase

See the real dependency web, find where logic is duplicated, and get an ordered, cycle-safe plan to consolidate it.

Review before you adopt

Clone a third-party repo read-only, fully map it, and get an adoption review before you commit to depending on it. codeweb never runs the target.
